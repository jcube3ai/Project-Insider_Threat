How to use this crosswalk: The Navigator layer JSON loads your ITM coverage directly into the official ATT&CK Navigator at mitre-attack.github.io/attack-navigator — go there, click the folder icon → Open Existing Layer → Upload File, then select attack-navigator-layer.json. Each highlighted cell is color-coded by ITM tactic. Direct = the ATT&CK technique is a precise match. Partial = conceptual overlap but ATT&CK models it differently or more narrowly. None = no ATT&CK equivalent exists — these are the unique value of this framework.
